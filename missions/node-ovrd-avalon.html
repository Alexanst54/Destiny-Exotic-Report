//NODE.OVRD.AVALON//

Year/Season: 2023 (Season of Defiance)

Location: Vex Network

Exotic Reward: Vexcalibur

Mission Overview

//NODE.OVRD.AVALON// is a secret mission set in the Vex Network. Guardians must solve puzzles, defeat Vex enemies, and face Variant Stalker Brakion to earn the Vexcalibur exotic glaive.

Key Bosses/Encounters

  • Variant Stalker Brakion, Repurposed Mind
